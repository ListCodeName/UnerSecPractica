Unidad 1: Selectores de Tags (Class, ID, Name, Custom)

Para poder aplicar estilos CSS o manipular elementos con JavaScript, necesitamos una forma de identificar o "seleccionar" los elementos HTML específicos. Aquí están los atributos más comunes para esto.

1. Selector de Clase (class)

El atributo class se utiliza para identificar un grupo de elementos. Múltiples elementos pueden tener la misma clase, y un elemento puede tener múltiples clases (separadas por espacio).

En CSS se selecciona con un punto (.nombreclase).

.destacado { color: blue; font-weight: bold; }
.error { color: red; }

2. Selector de ID (id)

El atributo id se utiliza para identificar un elemento único en toda la página. No debe haber dos elementos con el mismo ID.

En CSS se selecciona con un numeral/hash (#nombreid). Es muy útil también para enlaces internos (anclas).




  Este div tiene un ID único.




#unico {
    background-color: yellow;
    padding: 5px;
}
Este div tiene un ID único.

3. Atributo Name (name)

El atributo name se usa principalmente en elementos de formulario (input, select, textarea). Es el nombre que se envía al servidor junto con el valor del campo cuando se envía el formulario.

A diferencia del ID, varios elementos pueden compartir el mismo name (por ejemplo, en radio buttons).

4. Atributos Personalizados (data-*)

Los atributos data-* nos permiten almacenar información extra en los elementos HTML estándar, sin necesidad de usar hacks ni atributos no válidos. Son muy útiles para guardar datos que JavaScript necesita usar.

Pueden tener cualquier nombre siempre que empiecen con data-.
