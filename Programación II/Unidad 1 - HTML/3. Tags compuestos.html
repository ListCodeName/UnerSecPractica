Unidad 1: Tags Compuestos

Los tags compuestos son aquellos que requieren una estructura de varios elementos anidados para funcionar correctamente, como las tablas y las listas de selección.

1. Tablas (<table>)

Las tablas se usan para organizar datos en filas y columnas. No se deben usar para maquetar el sitio web.

  • <table>: Contenedor principal.
  • <tr> (Table Row): Define una fila.
  • <th> (Table Header): Define una celda de encabezado (negrita y centrada por defecto).
  • <td> (Table Data): Define una celda de datos estándar.
<table border="1">
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>Ana</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Carlos</td>
        <td>30</td>
    </tr>
</table>

2. Menús Desplegables (<select>)

El tag <select> crea una lista desplegable. Cada opción dentro se define con <option>.

Se pueden agrupar opciones con <optgroup>.

<label for="autos">Elige un auto:</label>
<select id="autos" name="autos">
    <optgroup label="Marcas Europeas">
        <option value="volvo">Volvo</option>
        <option value="bmw">BMW</option>
    </optgroup>
    <optgroup label="Marcas Americanas">
        <option value="ford">Ford</option>
        <option value="chevrolet">Chevrolet</option>
    </optgroup>
</select>

3. Listas de Descripción (<dl>)

Menos comunes pero útiles para glosarios o metadatos.

  • <dl>: Lista de descripción.
  • <dt>: Término a describir.
  • <dd>: Descripción o definición.
<dl>
    <dt>HTML</dt>
    <dd>Lenguaje de Marcado de Hipertexto.</dd>
    <dt>CSS</dt>
    <dd>Hojas de Estilo en Cascada.</dd>
</dl>
