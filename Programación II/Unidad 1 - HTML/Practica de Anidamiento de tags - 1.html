Práctica de Anidamiento de Tags - 1

Ejercicio 1: Detective de Errores

El siguiente código tiene 3 errores graves de anidamiento. Escríbelo correctamente en tu archivo.

<div>
    <p>Bienvenido a mi sitio
    <strong>Espero que te guste</p></strong>
    
    <ul>
        <p>Lista de cosas:</p>
        <li>Item 1
    </ul></li>
</div>

Ejercicio 2: La Matrioska

Te desafío a crear una estructura compleja respetando el anidamiento y la indentación.

Debes crear:
  1. Un <div> contenedor principal.
  2. Dentro, un <h2> con el título "Menú de Navegación".
  3. Debajo del título (aún dentro del div), una lista desordenada <ul>.
  4. Dentro de la lista, 3 elementos <li>.
  5. ¡Ojo! Dentro de CADA <li> debe haber un enlace <a>.
  6. Y dentro de cada enlace, el texto debe estar en negrita <strong>.
