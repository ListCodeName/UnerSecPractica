Práctica de Tags Compuestos - 1

Ejercicio 1: Horario de Clases

Crea una tabla que represente un horario escolar simple.

Debe contener:
  1. Una fila de encabezados (Lunes, Martes, Miércoles...).
  2. Al menos 3 filas con datos de materias (Matemáticas, Historia, etc.).
  3. Usa el atributo border="1" en la etiqueta table para ver las líneas.

Ejercicio 2: Selector de Comida

Crea un formulario para ordenar comida.

Debe contener:
  1. Un <select> para elegir el "Plato Principal".
  2. Dentro, usa <optgroup> para separar "Pizzas" de "Pastas".
  3. Agrega al menos 2 opciones de pizzas y 2 de pastas.
  4. Agrega otro <select> simple para elegir una bebida.
