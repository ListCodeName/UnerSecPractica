Práctica de Tags Más Usados - 1

Realiza los siguientes ejercicios en archivos separados o completa la estructura en tu editor de código favorito.

Ejercicio 1: Mi Perfil Personal

Crea una página HTML simple que represente un perfil personal.

Requisitos:
  1. Usa un <h1> para tu nombre completo.
  2. Usa un <p> para escribir una breve descripción sobre ti.
  3. Usa una lista desordenada <ul> para listar 3 pasatiempos o hobbies.
  4. Incluye una línea horizontal <hr> al final del contenido.

Ejercicio 2: Artículo de Blog

Estructura un pequeño artículo de noticias o blog.

Requisitos:
  1. Usa un <h2> para el título del artículo.
  2. Usa un <h3> para un subtítulo.
  3. Incluye al menos dos párrafos <p> de texto simulado (lorem ipsum).
  4. Inserta una imagen <img> relevante (puedes usar una de placeholder si no tienes una).
  5. Al final, agrega un enlace <a> que diga "Leer más..." y lleve a cualquier sitio web.
