Unidad 1: Botones e Inputs (Formularios)

Para interactuar con el usuario, HTML ofrece formularios y diversos elementos de entrada (inputs) y botones.

1. El elemento <form>

Es el contenedor principal para agrupar inputs y enviarlos a un servidor (aunque aquí solo veremos su estructura visual).

<form action="/enviar" method="POST"> ... </form>

2. Tipos de <input>

El tag <input> es el más versátil. Su comportamiento cambia según el atributo type.

Tipo Descripción
text Campo de texto de una sola línea.
password Campo de contraseña (oculta los caracteres).
email Valida que el texto sea un correo electrónico.
number Solo permite números.
date Selector de fecha.

Ejemplos de Texto

<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre" placeholder="Escribe tu nombre">

<label for="pass">Clave:</label>
<input type="password" id="pass" name="clave">


3. Selección: Radio y Checkbox

  • radio: Permite seleccionar una opción de un grupo (deben tener el mismo name).
  • checkbox: Permite seleccionar múltiples opciones.
<!-- Radio -->
<p>Género:</p>
<input type="radio" id="masc" name="genero" value="masculino">
<label for="masc">Masculino</label>
<input type="radio" id="fem" name="genero" value="femenino">
<label for="fem">Femenino</label>

<!-- Checkbox -->
<p>Intereses:</p>
<input type="checkbox" id="deporte" name="interes" value="deporte">
<label for="deporte">Deportes</label>
<input type="checkbox" id="musica" name="interes" value="musica">
<label for="musica">Música</label>

4. Botones

Hay dos formas principales de crear botones:

  1. <button>Texto</button>: La etiqueta moderna y flexible. Puede contener HTML dentro.
  2. <input type="button" value="Texto">: La forma antigua.
  3. <input type="submit" value="Enviar">: Botón especial para enviar formularios.
<button type="button" onclick="alert('Hola!')">Haz clic aquí</button>
<input type="submit" value="Enviar Formulario">
