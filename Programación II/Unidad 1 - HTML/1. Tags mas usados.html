Unidad 1: Tags Más Usados en HTML

En esta lección veremos los elementos fundamentales para construir una página web.

1. Estructura Básica

Toda página HTML debe tener una estructura mínima.

  • <!DOCTYPE html>: Define que es un documento HTML5.
  • <html>: El elemento raíz de la página.
  • <head>: Contiene metadatos (título, encoding) que no se ven en la página.
  • <body>: Contiene todo el contenido visible (texto, imágenes, etc.).
<!DOCTYPE html>
<html lang="es">
<head>
    <title>Mi Primera Página</title>
</head>
<body>
    Hola Mundo
</body>
</html>

2. Encabezados (Headings)

Se usan para títulos y subtítulos. Van del <h1> (más importante) al <h6> (menos importante).

<h1>Este es un título principal</h1>
<h2>Este es un subtítulo</h2>
<h3>Este es un encabezado menor</h3>

Este es un título principal

Este es un subtítulo

Este es un encabezado menor

3. Párrafos y Texto

El tag <p> define un párrafo. El navegador añade automáticamente un margen antes y después.

<p>Este es un párrafo de texto.</p>
<p>Este es otro párrafo.</p>

Este es un párrafo de texto.

Este es otro párrafo.

4. Enlaces (Links)

El tag <a> se usa para crear hipervínculos. El atributo href indica el destino.

<a href="https://www.google.com" target="_blank">Ir a Google</a>
Ir a Google

5. Imágenes

El tag <img> inserta imágenes. No tiene etiqueta de cierre. Requiere src (fuente) y alt (texto alternativo).

<img src="https://via.placeholder.com/150" alt="Imagen de ejemplo">

6. Listas

Listas desordenadas (<ul>) con viñetas y ordenadas (<ol>) con números. Los elementos son <li>.

<ul>
    <li>Elemento con punto</li>
    <li>Otro elemento</li>
</ul>

<ol>
    <li>Primer paso</li>
    <li>Segundo paso</li>
</ol>
  • Elemento con punto
  • Otro elemento
  1. Primer paso
  2. Segundo paso

7. Divisiones y Contenedores

<div> es un contenedor genérico de bloque (ocupa todo el ancho). <span> es un contenedor en línea (solo ocupa lo necesario).

<div style="background-color: lightblue; padding: 10px;">
    Esto es un div (bloque).
</div>
<p>Texto con una <span style="color: red; font-weight: bold;">palabra destacada</span> (span).</p>
Esto es un div (bloque).

Texto con una palabra destacada (span).
