Unidad 1: Profundizando en Imágenes

Aunque ya vimos el tag <img>, las imágenes en la web tienen muchos detalles importantes que debemos controlar para una buena experiencia de usuario.

1. Rutas (El atributo src)

El atributo src (source) le dice al navegador dónde encontrar la imagen. Existen dos tipos de rutas:

Rutas Absolutas

Es la dirección completa de una imagen en internet. Si el sitio original borra la imagen, dejará de verse en tu página.

<img src="https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png">

Rutas Relativas

Buscan la imagen dentro de las carpetas de tu propio proyecto. Son más seguras y rápidas.

  • src="foto.jpg": La imagen está en la misma carpeta que el archivo HTML.
  • src="img/foto.jpg": La imagen está dentro de una carpeta llamada "img".
  • src="../foto.jpg": La imagen está en la carpeta superior (atrás).

2. Accesibilidad (El atributo alt)

El atributo alt (texto alternativo) es OBLIGATORIO por buenas prácticas.

  • Se muestra si la imagen no carga (error de red o ruta incorrecta).
  • Es leído por lectores de pantalla para personas ciegas.
  • Ayuda a Google (SEO) a entender de qué trata la imagen.
<img src="perro.jpg" alt="Un perro golden retriever corriendo en el parque">

3. Dimensiones (width y height)

Siempre es recomendable definir el ancho (width) y alto (height) para evitar que el contenido "salte" mientras carga la imagen.

<img src="logo.png" alt="Logo de la empresa" width="200" height="100">

Nota: Si solo defines uno (ej. solo el ancho), el navegador calculará el otro automáticamente manteniendo la proporción.

4. Figuras semánticas (<figure>)

Cuando una imagen es importante para el contenido (como un gráfico o una ilustración con leyenda), usamos <figure> y <figcaption>.

<figure>
    <img src="https://via.placeholder.com/300x200" alt="Gráfico de ventas" width="300" height="200">
    <figcaption>Fig. 1: Crecimiento de ventas en 2023.</figcaption>
</figure>

5. Formatos Comunes

  • JPG / JPEG: Mejor para fotografías con muchos colores. Pierde calidad al comprimirse.
  • PNG: Soporta transparencia. Mejor para logos, gráficos o imágenes con texto.
  • SVG: Imágenes vectoriales (matemáticas). No pierden calidad al ampliarse. Ideales para logos e íconos.
  • GIF: Soporta animaciones simples. Limitado a 256 colores.
  • WebP: Formato moderno de Google. Alta calidad y bajo peso. (Recomendado).
