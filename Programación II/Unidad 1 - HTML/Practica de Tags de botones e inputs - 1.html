Práctica de Botones e Inputs - 1

Ejercicio 1: Formulario de Login

Diseña un formulario de inicio de sesión básico.

Debe contener:
  1. Un campo <input type="email"> para el correo electrónico, con un placeholder que diga "ejemplo@correo.com".
  2. Un campo <input type="password"> para la contraseña.
  3. Una etiqueta <label> para cada campo.
  4. Un botón <button> que diga "Ingresar".

Ejercicio 2: Encuesta de Preferencias

Diseña un formulario para conocer los gustos del usuario.

Debe contener:
  1. Un campo de texto para el "Nombre completo".
  2. Una pregunta "¿Qué color prefieres?" con 3 opciones usando radio buttons (Rojo, Verde, Azul). Recuerda usar el mismo atributo name para que sean excluyentes.
  3. Una pregunta "¿Qué frutas te gustan?" con 3 opciones usando checkboxes (Manzana, Banana, Naranja).
  4. Un botón de tipo submit para enviar.
