Unidad 3: Variables y Tipos de Datos

Abre la Consola del Desarrollador (F12 -> Pestaña Console) para ver los resultados.

1. Variables

Son contenedores para guardar información. Existen 3 formas de declararlas:

  • var: La forma antigua. (No recomendada hoy en día).
  • let: Para valores que pueden cambiar.
  • const: Para valores que NO deben cambiar (constantes).

2. Tipos de Datos Primitivos

  • String: Texto (entre comillas simples o dobles).
  • Number: Números (enteros o decimales).
  • Boolean: Verdadero (true) o Falso (false).
  • Undefined: Una variable declarada pero sin valor.
  • Null: Ausencia intencional de valor.
