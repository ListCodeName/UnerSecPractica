Unidad 3: DOM (Selección y Texto)

El DOM (Document Object Model) es la representación del HTML que Javascript puede modificar.

1. Seleccionar Elementos

Para modificar algo, primero hay que buscarlo.

  • document.getElementById("id"): El más rápido y específico.
  • document.querySelector(".clase" o "#id" o "tag"): Más flexible (como CSS).
