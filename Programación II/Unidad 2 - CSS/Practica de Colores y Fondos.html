Práctica de Colores y Fondos

Ejercicio 1: La Paleta de Pintor

Crea 3 cuadros (divs de 50x50px) y dales color de fondo usando los 3 métodos diferentes:

  1. Uno con nombre (ej. purple).
  2. Uno con Hexadecimal (ej. #2ECC71).
  3. Uno con RGB (ej. rgb(230, 126, 34)).

Ejercicio 2: El efecto fantasma (RGBA)

Crea un div con un texto y una imagen de fondo (puedes usar una URL de internet).

Encima de esa imagen (o dentro del div), pon otro div con texto.

  • Dale al div interior un color de fondo negro con 50% de opacidad (rgba(0,0,0,0.5)).
  • Ponle el texto en color blanco.

Esto se usa mucho para leer texto sobre imágenes.
