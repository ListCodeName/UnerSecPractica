Unidad 2: Formas de Incluir CSS

CSS (Cascading Style Sheets) es el lenguaje que usamos para dar estilo visual a nuestras páginas HTML. Existen tres formas principales de aplicarlo:

1. Estilo en Línea (Inline Style)

Se aplica directamente a un elemento HTML usando el atributo style. Es útil para pruebas rápidas pero no se recomienda para sitios reales porque es difícil de mantener.

<p style="color: red; font-size: 20px;">Texto rojo</p>

Texto rojo (Inline)

2. Estilo Interno (Style Tag)

Se escribe dentro de la etiqueta <style> en el <head> del documento HTML. Afecta solo a esa página específica.

<head>
    <style>
        h2 { color: blue; }
        .mi-clase { background-color: yellow; }
    </style>
</head>

3. Estilo Externo (External Stylesheet) (Recomendado)

El CSS se escribe en un archivo separado con extensión .css y se vincula al HTML usando la etiqueta <link>. Esto permite usar los mismos estilos en múltiples páginas.

<head>
    <link rel="stylesheet" href="mis-estilos.css">
</head>

Jerarquía (Cascada)

Si hay conflictos, el navegador decide qué estilo aplicar según la especificidad y el orden. Generalmente:

  1. Inline Style: Gana a casi todo (Mayor prioridad).
  2. Internal Style: Gana al externo si está definido después.
  3. External Style: Menor prioridad si hay conflictos directos, pero es la base de todo.
