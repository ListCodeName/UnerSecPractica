Práctica de Bordes

Ejercicio 1: Tarjeta de Presentación

Crea un div que parezca una tarjeta de visita.

  1. Dale un borde fino y gris (ej. 1px solid #ccc).
  2. Agrega padding para que el texto no toque el borde.
  3. Dale un poquitito de border-radius (ej. 5px o 8px) para que se vea moderna.
  4. (Opcional) Agrega una pequeña sombra (si te animas, o espera al tema de sombras).

Ejercicio 2: Foto de Perfil

Busca una imagen cuadrada en internet (o usa https://via.placeholder.com/150).

  • Aplícale border-radius: 50%; para hacerla redonda.
  • Ponle un borde grueso de color blanco (border: 5px solid white;) y un fondo de página que no sea blanco para que resalte.

Ejercicio 3: Botones Divertidos

Crea 3 botones con diferentes estilos de borde:

  1. Uno con borde discontinuo (dashed).
  2. Uno con borde solo abajo (border-bottom: 3px solid blue;).
  3. Uno con borde doble.
