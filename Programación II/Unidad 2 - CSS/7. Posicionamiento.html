Unidad 2: Posicionamiento

La propiedad position cambia cómo se comporta un elemento en el flujo del documento. Se usa junto con top, right, bottom, y left.

¡Soy fijo!

1. Static (Por defecto)

El comportamiento normal. top, left, etc. no le afectan.

2. Relative (Relativo)

Se mueve relativo a su posición original. El espacio que ocupaba originalmente se mantiene reservado (queda el hueco).

Relative

El texto sigue aquí, aunque la caja azul se movió 20px abajo y derecha.

3. Absolute (Absoluto)

Se mueve relativo al ancestro posicionado más cercano (usualmente un padre con position: relative). Sale del flujo normal (los otros elementos lo ignoran y pueden taparlo).

Soy el contenedor (padre) con position: relative.
Absolute

4. Fixed (Fijo)

Se queda fijo en la ventana del navegador, aunque hagas scroll. (Mira la caja amarilla abajo a la derecha de tu pantalla).

5. Z-Index

Controla la profundidad. Si dos elementos se superponen, el que tenga el z-index mayor se verá encima. Solo funciona en elementos con posición (no static).
