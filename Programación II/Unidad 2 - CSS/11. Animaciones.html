Unidad 2: Animaciones (@keyframes)

A diferencia de las transiciones (A -> B), las animaciones permiten control total de pasos intermedios y se pueden reproducir solas sin interacción del usuario.

1. @keyframes

Primero definimos "la película" usando @keyframes nombre-animacion.

@keyframes rebote {
  0% { transform: translateY(0); }
  50% { transform: translateY(-30px); }
  100% { transform: translateY(0); }
}

2. Propiedad animation

Luego "proyectamos" la película en un elemento.

animation: rebote 1s infinite alternate;
  • duration: Cuánto dura un ciclo.
  • iteration-count: Cuántas veces (1, 3, infinite).
  • direction: normal, reverse, alternate (va y vuelve).
