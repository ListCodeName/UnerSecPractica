Práctica de Sombras

Ejercicio 1: Botón Elevado

Crea un botón que parezca que está "flotando" sobre la página.

  1. Dale un color de fondo sólido (ej. azul).
  2. Dale un borde redondeado.
  3. Aplícale una sombra suave en la parte inferior (ej. 0 4px 6px rgba(0,0,0,0.3)).
  4. (Extra) Haz que cuando pases el mouse, la sombra se haga más grande (efecto de elevarse).

Ejercicio 2: Texto 3D / Retro

Crea un título grande (h1) de color brillante.

  • Usa text-shadow para duplicar el texto ligeramente desplazado y con un color más oscuro.
  • Prueba hacer varias sombras separadas por comas (ej. 2px 2px 0px black, 4px 4px 0px red).

Ejercicio 3: Tarjeta Minimalista

Crea un div blanco sobre el fondo gris de la página.

  • Para diferenciarlo, no uses bordes.
  • Usa solo una sombra muy sutil y difusa (ej. box-shadow: 0 10px 20px rgba(0,0,0,0.05);).
