Práctica de Inclusión de CSS

Ejercicio 1: El arcoíris inline

Crea 5 párrafos <p>. Usa el atributo style en cada uno para darles un color de texto diferente (rojo, naranja, amarillo, verde, azul).

Ejercicio 2: Estilos internos

En el mismo archivo HTML, agrega una etiqueta <style> en el <head>.

  • Define que todos los títulos h2 tengan el texto centrado (text-align: center;).
  • Define que el body tenga un color de fondo gris claro (background-color: #f0f0f0;).

Ejercicio 3: Archivo externo (Reto)

Crea un archivo nuevo llamado estilos.css en la misma carpeta.

  1. En el CSS, haz que los párrafos tengan un tamaño de fuente de 18px.
  2. En tu HTML, vincula este archivo usando <link rel="stylesheet" href="estilos.css">.
  3. Verifica que los párrafos cambien de tamaño.
