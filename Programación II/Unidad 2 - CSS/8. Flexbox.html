Unidad 2: Flexbox (Cajas Flexibles)

Flexbox es un módulo de diseño unidimensional para alinear y distribuir espacio entre elementos en un contenedor.

1. Activar Flexbox

Se aplica al padre (contenedor).

display: flex;

2. justify-content (Eje Principal - Horizontal por defecto)

Alinea los elementos horizontalmente.

  • flex-start: Al inicio (izquierda).
  • center: Al centro.
  • flex-end: Al final (derecha).
  • space-between: Separados al máximo entre ellos.
  • space-around: Espacio alrededor de cada uno.

3. align-items (Eje Secundario - Vertical por defecto)

Alinea los elementos verticalmente dentro del contenedor.

  • stretch: Se estiran para llenar el alto (default).
  • center: Se centran verticalmente.
  • flex-start: Arriba.
  • flex-end: Abajo.

4. gap (Espacio)

Define el espacio entre elementos flex. ¡Mucho mejor que usar márgenes!

gap: 20px;

5. flex-direction

Cambia la dirección del eje principal.

  • row: Fila (default, izquierda a derecha).
  • column: Columna (arriba a abajo).
