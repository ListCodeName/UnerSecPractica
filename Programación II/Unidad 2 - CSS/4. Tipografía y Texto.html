Unidad 2: Tipografía y Texto

El texto es el contenido principal de la web. CSS nos da control total sobre cómo se ve.

1. Familia de Fuentes (font-family)

Define la tipografía. Siempre es bueno poner una lista de opciones terminando en una genérica (serif, sans-serif, monospace).

Este es un texto Sans-Serif (sin remates, como Arial).
Este es un texto Serif (con remates, como Times New Roman).
Este es un texto Monospace (ancho fijo, como código).

2. Tamaño (font-size)

Controla el tamaño. Se pueden usar `px`, `em`, `rem`, `%`.

Texto pequeño (12px).

Texto grande (24px).

3. Peso (font-weight)

Controla el grosor de la letra.

  • normal (400)
  • bold (700)
  • Valores numéricos: 100 (muy fino) a 900 (muy grueso).

4. Decoración (text-decoration)

Muy usado para enlaces.

Texto subrayado (underline).

Texto tachado (line-through).

Enlace sin subrayado (none) - Muy común.

5. Sangría (text-indent)

Mueve la primera línea de un párrafo hacia la derecha.
