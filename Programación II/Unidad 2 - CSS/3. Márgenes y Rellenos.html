Unidad 2: El Modelo de Caja (Margen y Relleno)

Todo elemento en HTML es una caja rectangular. El espacio alrededor y dentro de esa caja se controla con margin y padding.

Márgen (Espacio externo)
Relleno (Espacio interno)
CONTENIDO

1. Padding (Relleno)

Es el espacio interno, entre el borde del elemento y su contenido. Si el elemento tiene color de fondo, el padding también se pinta de ese color.

padding: 20px; /* Todos los lados */
padding: 10px 20px; /* Vertical (10px) Horizontal (20px) */
padding-top: 10px; /* Específico */

2. Margin (Margen)

Es el espacio externo, entre el borde del elemento y los elementos vecinos. Es transparente (no tiene color).

margin: 20px; /* Todos los lados */
margin: 0 auto; /* 0 arriba/abajo, AUTOMÁTICO a los lados (Centra bloques) */
margin-bottom: 50px; /* Solo abajo */

3. Colapso de Márgenes

Nota curiosa: Si un elemento tiene margin-bottom: 20px y el de abajo tiene margin-top: 30px, el espacio entre ellos NO será 50px, sino 30px (el mayor de los dos). Esto se llama colapso de márgenes.
