Práctico N°2 — Tablas y Listas en HTML

Materia: Programación II   |   Tema: Tablas y Listas

Objetivo

Crear una tabla comparativa con cabeceras y filas de datos, y organizar información mediante listas ordenadas y desordenadas. Solo se permite el uso de HTML puro, sin hojas de estilo ni estilos en línea.

Contexto

El siguiente práctico simula una situación real en la que necesitás presentar información estructurada sobre tres sistemas operativos de uso común: Windows, Linux y macOS. Tu tarea es organizar esa información en una tabla comparativa y complementarla con listas que detallen las características de cada uno.

Consignas

Punto 1 — Crear el archivo de trabajo

Creá un nuevo archivo llamado tabla_so.html en tu carpeta de prácticos con la estructura HTML básica completa (doctype, html, head, body). Dentro del <head> colocá un título descriptivo para la página.

Punto 2 — Tabla comparativa

Dentro del <body>, creá una tabla que compare los tres sistemas operativos mencionados. La tabla debe tener las siguientes columnas:

  • Sistema Operativo
  • Desarrollador
  • Año de lanzamiento
  • Costo
  • Código fuente

Completá la tabla con los datos correctos para cada sistema operativo. Investigá los valores si no los conocés. La fila de cabeceras debe estar visualmente diferenciada del resto de las filas.

Punto 3 — Cabecera de la tabla

La fila de títulos de columna debe utilizar el elemento de encabezado de tabla correspondiente, de modo que el navegador la renderice con texto en negrita y centrado de forma nativa. No uses etiquetas de texto adicionales para lograr este efecto.

Punto 4 — Lista de ventajas

Debajo de la tabla, agregá un título que diga "Ventajas por sistema operativo". Luego creá una lista por cada sistema operativo con al menos 3 ventajas de cada uno. El tipo de lista debe transmitir que el orden de los ítems no importa.

Punto 5 — Pasos para instalar un sistema operativo

Al final del documento, agregá un título que diga "Pasos generales para instalar un sistema operativo". Listá al menos 5 pasos en el orden correcto en que deben realizarse. El tipo de lista debe reflejar que el orden sí importa.

Punto 6 (Adicional) — Pie de tabla

Agregá al final de la tabla una fila de cierre que ocupe todas las columnas y que diga: "Fuente: elaboración propia — [tu nombre] — [año]". Investigá cómo hacer que una celda se extienda a lo ancho de varias columnas.
